# ☕ Coffee Shop Sales Dashboard | Power BI

An interactive **Power BI dashboard** designed to analyze coffee shop sales data and provide valuable business insights through dynamic visualizations and KPIs. This project demonstrates how raw sales data can be transformed into meaningful reports that support data-driven decision-making.

## 📌 Project Overview

The dashboard provides a comprehensive analysis of coffee shop sales, helping users understand sales performance, product demand, and category-wise trends.

## 📊 Dashboard Features

* Total Sales Overview
* Total Quantity Sold
* Total Items Available
* Category-wise Sales Distribution
* Item-wise Quantity Sold Analysis
* Product Price Comparison (Small, Medium & Large)
* Interactive Filters for Category and Item Selection
* Dynamic Visualizations with Power BI

## 🛠️ Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX**
* **Microsoft Excel**
* **Data Cleaning & Transformation**
* **Data Visualization**

## 📈 Key Insights

* Identified the best-selling coffee shop products.
* Compared sales across different product categories.
* Analyzed quantity sold for individual menu items.
* Visualized pricing differences for Small, Medium, and Large sizes.
* Built interactive dashboards for better business reporting.
